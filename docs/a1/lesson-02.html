📖 Grammar

Questions = rising intonation. ვინ? = who, რა? = what, სად? = where. Identity: ეს ნინოა? — დიახ/არა.

  • Yes/no questions often need only rising intonation
  • ეს ნინოა? = Is this Nino?
  • დიახ = yes · არა = no
  • გაგიმარჯოს = informal reply to გამარჯობა

🇷🇺 Вопрос = интонация вверх. ვინ? = «кто?», რა? = «что?», სად? = «где?». Идентификация: «это Нино?» → ეს ნინოა?

გამარჯობა! ეს ნინოა?
Hello! Is this Nino?
💚 Easier than Russian: No «ты/вы» verb conjugation in greetings — გამარჯობა works for everyone.
