Full ვარ paradigm comes in L4 — start with არის for identifying things.

📖 Grammar

აი = here is / look. ეს არის X = this is X. ეს არ არის X = this is not X. Short form: nouns ending in -ი drop არის → just add -ა (კაფეა = it's a café).

  • აი + noun = here is / look at…
  • ეს არის X = this is X
  • ეს არ არის X = this is not X
  • Short form: კაფეა, ოფისია (= it is a café / office)

Most A1 nouns add -ს in dative. Used with მინდა, მაქვს, and -ში.

🇷🇺 აი ≈ «вот». ეს არის = «это есть». ეს არ არის = «это не». Краткая форма: после существительного на -ი вместо «არის» просто -ა (как у нас «это кафе» → კაფეა).

აი კაფე. ეს კაფეა.
Here is a café. This is a café.
💚 Easier than Russian: No gender on nouns — კაფე is just კაფე, no masculine/feminine endings.

📄 Short reading

აი ოფისი. ეს ოფისია. აი ტელეფონი და კომპიუტერი. ეს არ არის კაფე — ეს სუპერმარკეტია. აი სტუდენტი. აი ტურისტი ფოტოსთან.
Here is an office. This is an office. Here is a phone and a computer. This is not a café — this is a supermarket. Here is a student. Here is a tourist by the photo.
