📖 Grammar

Obligation: აუცილებელია + verb. Instruction language uses polite imperative forms. Question particle ხომ softens urgent questions.

🇷🇺 Обязанность: აუცილებელია + глагол. Инструкции: вежливый императив. Частица ხომ смягчает вопрос.

თუ საფრთხეა, აუცილებელია დაუყოვნებლივ დარეკოთ 112-ზე.
If there is danger, you must call 112 immediately.
💚 Easier than Russian: 112 is 'ას თორმეტი' — same emergency number logic as 911.

📄 Short reading

ავარია მოხდა! დავურეკე 112-ს. ოპერატორმა კითხვები დამისვა: სად ხარ? რა მოხდა? მისამართი ვთქვი და დახმარება მალე მოვიდა.
An accident happened! I called 112. The operator asked questions: where are you? what happened? I gave the address and help came quickly.
